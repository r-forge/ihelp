기본 그래프 - plot() 함수사용시 그래프에 추가정보 삽입방법

Sys.setlocale("LC_CTYPE", "Korean_Korea")

기본 plot()함수를 사용할때 추가정보를 출력된 그래프에 첨가할때 아래와 같이 할수있다. 여기서 추가정보는 미리 출력된 그래프에 보태짐으로, 항상 plot()으로 먼저 그래프를 출력시킨 상태에서 아래의 함수들을 사용한다.

데이터 점을 추가할때

출력된 그래프 위에 데이터점을 추가할때 보통 points()함수를쓴다. 이 방법으로 다른 데이터 소스에있는 데이터점을 추가시킬수도 있으나 points() 함수는 기존의 plot()으로 출력된 x와 y축의 범위내에서만 데이터점이 추가되므로, 보통은 같은 데이터내에서있는 특정 데이터점들을 확인시켜주는 방법으로 많이 쓰인다. 특히 데이터 점들이 많을경우, 기존 plot()에서 색깔을 다르게 정해줄때 겹치는 점들이 잘 나타나지 않기때문에 points()를 사용하면 특정 데이터 점들이 더 잘 확인될수있다.

아래는 iris데이터의 기본 plot()이다. 

plot(Sepal.Width ~ Sepal.Length, iris, col = "grey", pch = 16)

아래는 위의 그래프에서 setosa 종들을 따로 확인시켜준다.

Ir1 <- iris[which(iris$Species == "setosa"), ]
plot(Sepal.Width ~ Sepal.Length, iris, col = "grey", pch = 16)
points(Ir1$Sepal.Length, Ir1$Sepal.Width, col = "red", pch = 16)

선을 추가할때

출력된 그래프 위에 선을 추가할때 보통 lines()함수를쓴다. 이 함수는 출력된 그래프의 점들을 연결시켜주기도하고, 최적 라인등을 추가할때 유용하게 사용된다.

아래는 산점도 그래프의 점들을 lines()를 사용하여 연결한 그래프이다. 이 함수에 사용되는 다른 매개변수들은 많지만 주로 lty를 사용하여 실선이나 다른 종류의 점선을 선택할수있다. 그리고, col을 사용하여 색상을 선택한다. 

x <- c(1:10)
y <- 1/x

plot(y ~ x, col = "red", pch = 16)
lines(y ~ x, col = "blue", lty = 1)

아래에서는 cars데이터의 기본 산점도 그래프에 선형회귀 (linear regression)를 사용해서 얻은 최적라인(초록색 선)과 lowess()를 사용해서 얻은 부드러운(smoothed) 최적라인(붉은색 선)을 추가하였다. 

LM <- lm(dist ~ speed, cars)

plot(cars, col = "steelblue", pch = 16)
lines(cars[, "speed"], LM$fitted.values, col = "green", lty = 1)
lines(lowess(cars), col = "red", lty = 1)

글씨를 추가할때

아래는 위에서 출력된 cars 그래프에 선형회귀를 사용해서 얻은 최적라인(초록색 선)과 lowess()를 사용해서 얻은 부드러운(smoothed) 최적라인(붉은색 선)에 해당되는 라벨을 text()함수를 사용해서 삽입하였다. 여기서 라벨의 위치를 그래프의 x, y 좌표를 사용해서 정해준다. 이 함수에 사용되는 매개변수는 plot()에 사용되는 것과 비슷하며 주로 cex로 글씨의 크기를 조절하고 col로 색을 정해준다.

LM <- lm(dist ~ speed, cars)

plot(cars, col = "steelblue", pch = 16)
lines(cars$speed, LM$fitted.values, col = "green", lty = 1)
text(cars[nrow(cars), "speed"] - 2, LM$fitted.value[nrow(cars)] + 2, label = "선형회귀", 
    col = "green", cex = 0.7)
lines(lowess(cars), col = "red", lty = 1)
text(lowess(cars)$x[nrow(cars)] - 2, lowess(cars)$y[nrow(cars)] - 17, label = "lowess", 
    col = "red", cex = 0.7)

text()를 사용해서 그래프의 점들대신 라벨로 표시할수도있다. 

plot(uptake ~ conc, CO2, type = "n")
text(CO2$conc, CO2$uptake, label = CO2$Type, col = as.numeric(factor(CO2$Treatment)), 
    cex = 0.7)

아래는 text()를 사용해서 그래프의 점들옆에 라벨을 삽입할때이다. 여기서 pos라는 매개변수를 사용하는데 그 값은 아래와 같다.
1=점의 아래편
2=점의 왼편
3=점의 위편
4=점의 오른편 

plot(uptake ~ conc, CO2, type = "p", col = "steelblue", cex = 1.3)
text(CO2$conc, CO2$uptake, label = CO2$Type, col = as.numeric(factor(CO2$Treatment)), 
    cex = 0.7, pos = 4)

아래는 mtext()함수를 써서 그래프 밖에있는 마진에 라벨을 삽입하는 예이다. 

plot(uptake ~ conc, CO2, type = "p", main = "CO2", col = "steelblue", cex = 1.3)
mtext("이산화탄소", 3, line = 0.5, cex = 0.8)

기준선을 추가할때

여기서 소개할 함수는 abline()이다. 이 함수는 보통 기준선을 그릴때 쓰지만 직선을 그리는 함수로 보면 되겠다. abline()으로 가로선과 세로선을 그릴수도있고 주어진 경사와 절편을이용해서 직선을 그릴수도있다. 특히 위에서 처럼 선형회귀로 얻은 예측값을 그래프로 옮길때 간편하게 쓸수있다. 

LM <- lm(dist ~ speed, cars)

plot(cars, col = "steelblue", pch = 16)
abline(v = 20, lty = 2)
text(x = 21, y = 20, label = "세로선", cex = 0.7)
abline(h = 100, lty = 2)
text(x = 23, y = 105, label = "가로선", cex = 0.7)
abline(a = 10, b = 2, col = "red", lty = 3)
text(x = 21, y = 56, label = "경사 2, 절편 10", col = "red", cex = 0.7)
abline(LM, col = "green", lty = 1)
text(x = 22, y = 80, label = "선형회귀선", cex = 0.7, col = "green")

기호 일람표(legend)를 추가할때

그래프를 그릴때 특히 한개이상의 기호(key)를 사용했을때 기호일람표를 삽입하게된다. 기본 plot()에서는 legend()함수를 사용하여 사용된 기호의 뜻을 표시한다.

legend()함수에서 일람표의 위치를 정해줄때 x,y 좌표를 이용해서 위치를 정해줄수도있고 아래와 같이 글로 위치를 정해줄수도있다.

“top” - 상단
“right” - 우측
“left” - 좌측
“bottom” - 하단
“topright” - 우측상단
“topleft” - 좌측상단
“bottomright” - 우측하단
“bottomleft” - 좌측하단

아래의 예에서는 4가지 기호들이 모양과 색깔로 각기 나타내는 다른 내용들을 표시한다.

plot(uptake ~ conc, CO2, col = as.numeric(factor(CO2$Treatment)), pch = as.numeric(factor(CO2$Type)))
legend("topleft", legend = c(levels(factor(CO2$Treatment)), levels(factor(CO2$Type))), 
    col = c(1, 2, "grey", "grey"), pch = c(15, 15, 1, 2), cex = 0.8, pt.cex = 0.8)

locator() 사용하기

기호일람표나 라벨을 삽입할때 locator()를 사용하면 출력된 그래프위에서 마우스로 클릭을 하면서 원하는 위치에 삽입할수있다. 여기서 locator()함수안에 들어가는 숫자는 삽입할 갯수인데 보통 기호 일람표를 삽입할때는 1을 쓰고, 라벨을 추가할때 원하는 갯수를 정해준다. 

LM<-lm(dist~speed,cars)

plot(cars,col="steelblue",pch=16)
lines(cars[,"speed"],LM$fitted.values,col="green",lty=1)
lines(lowess(cars),col="red",lty=1)
legend(locator(1),legend=c("LM","lowess"),col=c("green","red"),lty=1)
text(locator(4),label="cars",cex=1)

제목을 추가할때

plot()함수내에서 보통 제목이나 부제목을 삽입하지만 title()함수로도 삽입할수있다. plot()함수가 너무 길어질때 유용하다.

plot(uptake ~ conc, CO2, type = "p")
title(main = "이산화탄소 섭취", sub = "농도에따라 변하는 섭취율")

데이터 점들 식별하기

출력된 그래프위에서 마우스 클릭으로 점들의 나타내는 값을 보여줄때 identify()함수를 사용한다. 그래프위에서 원하는 점들을 클릭한후 esc를 누르면 눌렀던 위치에 있는 점들의 값이 나타내어진다. 아래의 예에서는 identify()를 사용하여 CO2데이터의 Type이 나타내어지게 작성하였다.

plot(uptake~conc,CO2,type="p")
identify(CO2[,c("conc","uptake")],label=CO2$Type)

그래프내에서 면적 표시하기

출력된 산점도 그래프의 점들을 연결하여 면적을 표시할때 polygon()함수를 쓴다. 아래의 예에서는 출력된 그래프안에서 의 실제 y값과 선형회귀 모델로 계산한 예측값의 오차범위를 노란색으로 표시하였다. 

Dat_QNC <- subset(CO2, Plant == "Qn1" & Type == "Quebec" & Treatment == "nonchilled")

LM <- lm(uptake ~ conc, Dat_QNC)

P <- predict(LM, data.frame(conc = c(160, 770)))

A <- rbind(cbind(conc = c(160, 770), uptake = P), Dat_QNC[which(Dat_QNC$conc >= 
    160 & Dat_QNC$conc <= 770), c("conc", "uptake")])

A <- A[order(A$conc), ]

plot(uptake ~ conc, Dat_QNC, type = "l")
abline(LM, lty = 3)
polygon(A$conc, A$uptake, col = "yellow", border = "green")

축을 추가할때

plot()함수내에서 축을 표시하지않고 그래프 출력후에 축을 삽입하는 방법으로 axis()함수가 있다. 이 방법의 장점은 축의 위치를 그래프의 사면에 다 정할수있다는 것인데 아래의 숫자로 위치를 정해준다.

1=하단
2=좌측
3=상단
4=우측 

x <- c(1:10)

par(oma = c(1, 2, 1, 3))
plot(1/x ~ x, col = "red", type = "l", lty = 1, yaxt = "n", ylab = "1/x")
par(new = TRUE)
plot(x^2 ~ x, col = "blue", type = "l", lty = 1, yaxt = "n", ylab = "")
axis(2, at = c(1, seq(20, 100, by = 20)), label = seq(0, 1, by = 0.2), pos = c(0.6, 
    0), lty = 1, col = "red", las = 2, cex = 0.6)
axis(4, at = c(1, seq(20, 100, by = 20)), label = NULL, pos = c(10.4, 0), lty = 1, 
    col = "blue", las = 2, cex = 0.6)
mtext("x^2", 4, line = 3)
