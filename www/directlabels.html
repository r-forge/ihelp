directlabels 팩키지 소개

보통 그래프안에 다수의 범주가 보여질때 기호 일람표(legend)를 사용하는데, 특히 그 수가 많을경우 그래프안에 상응하는 선이나 위치에 범주값을 표시하는 방법이 유용할때가 있다. 예를 들어 다른 색으로 범주값을 표시할때 비슷한 색을 피하려면 보통 선택할수있는 색깔이 5-6개정도로 제한되며, 점들의 모양을 다르게 설정하는것도 4-5개이상이면 구분하기 힘들어진다. 색깔과 모양을 결합해서 사용하더라도 7-8개이상의 조합을 사용하면 눈으로 식별하기가 어려워진다. 이럴경우 다른 색이나 모양을 사용할때 범주값을 그래프안에 상응하는 위치에 표시하면 복잡한 그래프도 이해하기 쉬워질수있고 비슷한 색이라도 구분이 쉬워진다.

보통 text()라는 함수로 그래프안에 라벨을 붙이는데, directlabels라는 팩키지를 사용하면 훨신 더 간단하게 자동으로 그래프내에 지정된 범주값에 상응하는 표시를 할수있도록 해준다.

아래의 예들은 directlabels 팩키지를 이용하여 표식을 그래프안에 넣어줌으로서 다른 도표없이 그래프만으로 간단한 정성분석이 쉽게됨을 보여준다. 

library(directlabels)

이 문서에서 예로 사용될 데이터는 아래의 사이트에서 nybirths.dat 이라는 1946년에서 1959년사이 뉴욕에서 매월 별로 출생횟수를 기록한 데이터이다.

http://robjhyndman.com/tsdldata/data/

우선 연도별로 데이터의 누적합계를 계산하여 몇년도가 출생률이 더 높았는지 알아볼때 directlabels를 사용한 방법이다. 

births <- scan("http://robjhyndman.com/tsdldata/data/nybirths.dat")

my <- seq(from = as.Date("01-Jan-1946", format = "%d-%b-%Y"), by = "months", 
    length = 168)

cs <- matrix(births, 12, 14)
cs <- apply(cs, 2, cumsum)

births_cs <- data.frame(Year = format(my, "%Y"), Month = format(my, "%b"), birth_rate = births, 
    Birth_cs = as.vector(cs))

여기서 x축이 월인데 숫자가아닌 영문으로 나타낼경우 R에서 자동으로 factor를 알파벳순으로 정한다. 그러므로 월의 순서가 뒤죽박죽되기 때문에 아래의 방법으로 올바른 서열을 정해준다. 

births_cs$Month <- factor(births_cs$Month, levels = format(my[1:12], "%b"))

그리고, 출생횟수의 누적합계를 제곱한이유는 그래프상에서 각 연도별차이가 좀더 뚜렸히 보여지게 하기위함이며 특별한 다른 이유는 없다.

아래의 그래프에서 기본적으로 추출할수있는것은 뉴욕에서는 1956년부터 1959년까지 출생수가 상대적으로 매우 높았음을 볼수있다. 이는 세계 2차대전이 끝난후 뉴욕이 빠르게 성장한 시기와 맞물린다. 아래 소스 참조.

http://en.wikipedia.org/wiki/History_of_New_York_City_(1946%E2%80%9377)

direct.label(xyplot((Birth_cs)^2 ~ Month, births_cs, group = births_cs$Year, 
    type = "l", xlab = "월", ylab = "(출생횟수)^2"))

아래는 월간 출생률의 밀도를 연도별로 비교한것이다. 40년대에 비해 50년대 특히 50년대 후반에 출생률이 명확하게 높아진것을 볼수있다. 특이한 점은 1946년도의 출생률이 나머지 40년대보다 높게 (50년대 초반과 비슷하게) 나온다는 점이다. 이것은 전쟁이 끝난 직후(1년후)라는것이 높은 출생률과 관련이 있을 가는성이 높고, 50년대의 높은 출생률은 좋아진 경제성장의 결과일 가능성이 높다. 

direct.label(densityplot(~birth_rate, births_cs, group = births_cs$Year, xlab = "월간 출생률", 
    ylab = "밀도", n = 100))
