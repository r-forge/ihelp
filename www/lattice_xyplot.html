고등의 그래프를 그리는 팩키지중 xyplot이나 density plot 또는 contour map이나 3차원 contour map등등을 그릴수있게 해주는 팩키지에 lattice 팩키지가 있다. 여기서는 우선 가장 보편적으로 사용할수있는 xyplot() 함수를 소개하고자 한다. 기본 plot()과 비슷한면도 있지만 더욱 복잡한 그래프도 상대적으로 간단한 코드를 사용해서 그릴수있다.

우선 사용가능한 그래프 샘플들을 보려면 기본 plot() 처럼 demo() 함수를 사용하여 저장되어있는 샘플 그래프들을 볼수있다.

demo(lattice)

아래는 기본적인 xyplot 그래프이다.

library(lattice)

xyplot(incidence ~ year, melanoma)

위에 그려진 그래프의 형을 산포도 그래프에서 다른것으로 바꾸어 주려면 기본그래프와 같이 'type'이라는 매개변수를 지정해준다. 예로 위의 그래프를 산포도에서 선형 그래프로 바꾸려면 아래와같이 한다. 

xyplot(incidence ~ year, melanoma, type = "l")

'type'이라는 매개변수값에는 기본 그래프보다 더 많은 선택항목들이 있으며, 이 선택항목들을 사용하여 그래프의 추가적인 출력들을 더욱 손쉽게 만들수있다. 그리고 한개 이상의 항목을 선택해서 여러가지 그래프 형을 동시에 출력시킬수도 있다.

예로 처음 그래프에 선형회귀로 구한 최적라인을 추가한 그래프를 그리려면 아래와같이 한다.

xyplot(incidence ~ year, melanoma, type = c("p", "r"))

loess 선을 추가한 그래프는 아래와같이 한다.

xyplot(incidence ~ year, melanoma, type = c("p", "smooth"))

스플라인 선을 추가한 그래프는 아래와같이 한다.

xyplot(incidence ~ year, melanoma, type = c("p", "spline"))

그리드를 추가하려면 아래와같이 한다.

xyplot(incidence ~ year, melanoma, type = c("p", "spline", "g"))

그래픽요소를 바꾸어 주는것과 제목및 부제목을 입력하는것은 plot()함수와 같다. 다른점은 제목이나 부제목의 글씨크기나 색등의 스타일을 바꾸려면 par.setting을 사용하여야 한다는 것이다.

xyplot(incidence ~ year, melanoma, type = c("p", "r"), cex = 1.2, pch = 16, 
    col = "red", main = "Melanoma Incidents by Year", sub = "melanoma incidence over time", 
    par.settings = list(par.main.text = list(cex = 1.8, col = "blue"), par.sub.text = list(cex = 1.2, 
        col = "green"), par.xlab.text = list(cex = 0.8, col = "purple"), par.ylab.text = list(cex = 0.8, 
        col = "steelblue")))

다수의 창(pane)을 사용해서 xyplot 그리기

하나의 데이터소스안에서 특정 변수를 기준으로 변수값에 해당되는 다른 그래프들을 동시에 다수의 창에서 출력시킬때 아래와 같이한다. 만약 수치형변수를 기준으로 창을 나눌때는 as.character()를 사용하여 범주형변수로 먼저 바꿔준다.

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"))

위의 그래프는 각 창의 x-, y-축들이 같은 스케일로 되어 있는데, 만약 각 창마다 포함된 데이터값에 맞추어서 스케일을 바꾸려면 아래와 같이 scales를 “free"라고 지정해준다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = "free")

만약 x축의 스케일만 포함된 데이터값에 맞추어서 스케일을 바꾸려면 아래와 같이한다. y축만 바꾸려면 아래의 함수에서 y="free"라고 바꾸어준다.

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = list(x = "free"))

만약 x나 y축의 스케일을 임의대로 정해주려면 아래와 같이 scales를 지정해준다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = list(x = "free", 
    y = list(limits = c(100, 800))))

창이 나열되어있는 위치를 바꾸려면 아래와 같이 layout을 사용한다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = "free", 
    layout = c(1, 2))

창의 제목의 글씨 크기를 바꾸려면 아래와 같이 par.strip.text의 cex를 변경해준다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = "free", 
    par.strip.text = list(cex = 0.6))

창의 제목이 쓰여진 틀의 크기를 바꾸려면 par.settings안에서 layout.heights를 아래와 같이 지정해준다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = "free", 
    par.settings = list(layout.heights = list(strip = 2)))

x,y 축의 표기 방향과 크기를 바꾸려면 아래와 같이 한다.

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), scales = list(x = list(cex = 0.8, 
    rot = 45), y = list(cex = 1.2, rot = 0)), par.settings = list(layout.heights = list(strip = 2)))

그래프내에서 그룹을 정해줄때

데이터 변수중 범주형변수가 있을때, 데이터의 수치형변수들을 사용해 그려진 산포도 그래프안에 범주값들을 그룹으로 표시해서 2차원의 도표에서 3차원의 효과를 나타낼수있다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), groups = CO2$Treatment)

위의 그래프에서 그룹으로 정해진 변수값들을 기호일람표로 표시할때 아래와 같이한다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), groups = CO2$Treatment, 
    auto.key = TRUE)

기호일람표의 위치나 내용을 바꿀때 아래와 같이한다. 

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), pch = 16, groups = CO2$Treatment, 
    auto.key = list(space = "right", text = c("non-chilled loess", "chilled loess"), 
        cex = 0.7, size = 1.7, lines = TRUE, points = FALSE))

위의 그래프처럼 점들이 겹칠경우 alpha 매개변수를 지정하여 투명도를 변경할수도 있다. 여기서 alpha는 0에서 1사이의 숫자로 수가 커질수록 투명도가 낮다.

xyplot(conc ~ uptake | Type, CO2, type = c("p", "smooth"), pch = 16, groups = CO2$Treatment, 
    alpha = 0.5, auto.key = list(space = "right", text = c("non-chilled loess", 
        "chilled loess"), cex = 0.7, size = 1.7, lines = TRUE, points = FALSE))

xyplot()함수 안에서 사용되는 매개변수들의 종류와 옵션들이 매우 많으므로, 위에서 설명하지 않은 매개변수들은 아래의 함수를 사용하여 찾을수있다.

trellis.par.get()

각 창에 있는 그래프에 추가요소나 추가 그래프를 더해줄때

plot()과 마찬가지로 xyplot에서도 다른 데이터소스를 사용한 그래프를 기존 출력된 그래프안에 겹쳐서 출력 시킬수있다. 이런 경우 panel.으로 시작하는 함수를 사용하는데, 아래의 예는 산포도 그래프안에 abline을 사용해서 기준선을 삽입하는 방법이다. 

Dat <- split(CO2, CO2$Plant)
xyplot(conc ~ uptake | Type, CO2, type = "p", groups = CO2$Treatment, pch = 16, 
    alpha = 0.5, panel = function(x, y, ...) {
        panel.xyplot(x, y, ...)
        panel.abline(v = 30, lty = 3, col = "red")
    })

각 창안에서 그래프요소들을 데이터내에있는 매개변수에 맞추어서 변형시키려면 아래와같이 'subscript'를 사용한다.

Dat <- split(CO2, CO2$Plant)
xyplot(conc ~ uptake | Type, CO2, type = "p", K = CO2$Treatment, groups = CO2$Plant, 
    panel = function(x, y, K, groups, ..., subscripts) {
        panel.xyplot(x, y, pch = 16, col = groups, cex = ifelse(K[subscripts] == 
            "chilled", 2, 1), alpha = 0.5)
        panel.abline(v = 30, lty = 3, col = "red")
    })
