주성분 분석 (Principal Component Analysis) – 주성분 추출에 사용되는 방법

흔히 주성분 분석이라하면 주어진 변수들을 가장 잘 설명할수있는 직교되는 새로운 축을 찾는 방법을 말한다. 여기서 새로운 축을 찾는 이유는 보통 차원축소를 하기위함이며 이 방법이 응용되는 범위는 다양하고 다른 응용 방식들은 계속해서 개발이되고있다. 한 예로 상대적으로 최근에 주성분 분석이응용되는 분야에는 이미지 분석(image analysis) (예: 얼굴인식 (facial recognition)) 등이 있다.

R 에서 주성분분석을 사용하고 응용하는 방법은 R이가진 특성상 사용자에의해 다양하게 이루어진다.

1. princomp() 명령어 사용

보편적으로 사용이되는 방법들에는 우선 princomp()이 있다. 다른 특별한 package를 사용하지않고 base package에 있는방법중에 하나로 사용하기 매우 간단하며 출력 속도또한 빠르다.

iris 데이터중 수치변수들만 미리 선택하는 방법

Dat <- iris
princomp(Dat[, 1:4])

## Call:
## princomp(x = Dat[, 1:4])
## 
## Standard deviations:
## Comp.1 Comp.2 Comp.3 Comp.4 
## 2.0494 0.4910 0.2787 0.1539 
## 
##  4  variables and  150 observations.

공식을 사용하는 방법

princomp(~Sepal.Length + Sepal.Width + Petal.Length + Petal.Width, Dat)

## Call:
## princomp(formula = ~Sepal.Length + Sepal.Width + Petal.Length + 
##     Petal.Width, data = Dat)
## 
## Standard deviations:
## Comp.1 Comp.2 Comp.3 Comp.4 
## 2.0494 0.4910 0.2787 0.1539 
## 
##  4  variables and  150 observations.

위에서 보여지듯이 사용하는 방식은 달라도 결과는 같다.

여기서 기본적으로 princomp()은 데이터를 covariance matrix로 변화시키는데 만약 변수중 variance가 0인 변수가있을때 (즉 상수 - constant variable 일때), princomp()는 해당하는 주성분을 기본 축중 하나로 동일시 취급한다. 즉 변환이 되지 않는다.

혹시 correlation matrix를 사용해서 주성분을 추출할경우 princomp() 명령어를 쓰려면 아래와같이 하면 된다.

princomp(Dat[, 1:4], cor = TRUE)

## Call:
## princomp(x = Dat[, 1:4], cor = TRUE)
## 
## Standard deviations:
## Comp.1 Comp.2 Comp.3 Comp.4 
## 1.7084 0.9560 0.3831 0.1439 
## 
##  4  variables and  150 observations.

여기서 주의할점은 correlation matrix를 사용할때 데이터에 상수가 존재하면 에러가 발생하므로 미리 제거를 한후 주성분을 추출하여야한다.

주성분 축에 해당하는 vector 즉 eigen vector를 찾으려면 $loading 으로 출력하면 된다.

princomp(Dat[, 1:4])$loading

## 
## Loadings:
##              Comp.1 Comp.2 Comp.3 Comp.4
## Sepal.Length  0.361 -0.657  0.582  0.315
## Sepal.Width         -0.730 -0.598 -0.320
## Petal.Length  0.857  0.173        -0.480
## Petal.Width   0.358        -0.546  0.754
## 
##                Comp.1 Comp.2 Comp.3 Comp.4
## SS loadings      1.00   1.00   1.00   1.00
## Proportion Var   0.25   0.25   0.25   0.25
## Cumulative Var   0.25   0.50   0.75   1.00

그리고 변환된 주성분 값 (즉 principal components)을 찾으려면 아래와 같이 $score 로 출력하면된다.

head(princomp(Dat[, 1:4])$score)

2. cmdscale() 명령어 사용

우선 이 방법을 설명 하기전에 위에서 거론된 차원축소에서 어떻게 주성분분석이 이용되는지를 설명할 필요가있겠다. 차원축소란 주어진 데이터에 있는 n개의 변수들을 주성분으로 변환하여 n보다 적은 수의 차원으로 데이터를 전환시키는 방법이다. 이 주성분들은 기존의 변수들을 설명할수있는 새로운 축으로 변환된것이며 그 수가 n개 미만이 되더라도 기존의 데이터를 잘 설명할수있다. 여기서 흔히 쓰는 차원축소 방법중에 하나가 다차원척도법 (multidimensional scaling)이다. R에서는 cmdscale()이라는 명령어를 쓰는데 이 명령어로 데이터를 전환했을 경우 변환된 값은 주성분분석을 이용해 얻어진 변수값과 같아야 한다. 이것은 Covariance를 사용한 주성분값은 Euclidean distance를 사용한 다차원척도법 값과 동일하기 때문이다. 자세한 설명은 여기를 참조.

cmdscale() 도 princomp()와 같이base package 이며, 쉬운 비교를위하여 위에서 사용했던 iris 데이터를 다시 사용해서 cmdscale()로 주성분분석을 시도하겠다. 다른점은 cmdscale()은 데이터를 우선 distance matrix로 바꾸어주어야한다. dist()이란 명령어로 간단하게 바꿀수있다

D <- dist(Dat[, 1:4])
head(cmdscale(D, k = 4))

##        [,1]    [,2]     [,3]      [,4]
## [1,] -2.684  0.3194  0.02791  0.002262
## [2,] -2.714 -0.1770  0.21046  0.099027
## [3,] -2.889 -0.1449 -0.01790  0.019968
## [4,] -2.745 -0.3183 -0.03156 -0.075576
## [5,] -2.729  0.3268 -0.09008 -0.061259
## [6,] -2.281  0.7413 -0.16868 -0.024201

여기서 k 값은 축소하고 싶은 차원의 갯수를 말한다. 예로 4차원의 데이터를 3차원으로 줄이고 싶을땐 k=3라고 설정한다. 그러므로 4차원의 데이터의 모든 주성분값을 알고 싶으면 k=4라고 설정하면 된다. 여기서 출력된 주성분 변수들은 일치하는 eigen value가 작아지는 순서대로 나열되있다. 즉 마지막으로 나열된 주성분 변수는 가장 작은 eigen value를 갖고있고 차원을 축소시키는 과정에서 일차적으로 소거된다.

비교하기 쉽게 두 방법의 출력결과를 보여주는 그래프들을 양옆으로 놓아보겠다. 

par(mfrow = c(1, 2))
plot(princomp(Dat[, 1:4])$score[, 1:2], main = "princomp()", xlab = "PC1", ylab = "PC2")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)
plot(cmdscale(dist(Dat[, 1:4]), k = 4)[, 1:2], main = "cmdscale()", xlab = "PC1", 
    ylab = "PC2")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)

princomp()의 결과물과 cmdscale()의 결과물이 x축을 기준으로 미러 이미지 형태로 나왔음을 알수있다.

3. 수학적으로 푼 코드

보통 간단한 명령어로 답을 얻는것이 바람직하나 때로는 수학적인 방법으로 풀어서 코드를 작성하는것이 필요할때가 있다. 이것은 기존의 방식에서 벗어난 변형된 방식을 시도할때 유용하게 쓰여진다.

Dat <- iris[, 1:4]

혹시 있을지모르는 상수들 제거

V <- apply(Dat, 2, var)
Input <- Dat[, which(V > 0)]

각 변수들의 평균값을 0으로 변환

Mean <- apply(Input, 2, mean)
AdjD <- t(apply(Input, 1, function(x) (x - Mean)))

Covariance matrix 만들기. 여기서 cov() 을 사용하나 var() 을 사용하나 같은 결과.

CovA <- var(AdjD)

eigen vector구하기. 위에 covariance는 대칭하는 matrix 가 형성되므로 symmetric=TRUE 라 설정을 해야지 대칭되는 값을 중복해서 사용하지 않는다

EigVec <- eigen(CovA, symmetric = TRUE)

eigen vector를 이용한 주성분 추출

temp <- t(EigVec[[2]]) %*% t(AdjD)
ConvD <- t(temp)
head(ConvD)

##        [,1]    [,2]     [,3]      [,4]
## [1,] -2.684  0.3194 -0.02791  0.002262
## [2,] -2.714 -0.1770 -0.21046  0.099027
## [3,] -2.889 -0.1449  0.01790  0.019968
## [4,] -2.745 -0.3183  0.03156 -0.075576
## [5,] -2.729  0.3268  0.09008 -0.061259
## [6,] -2.281  0.7413  0.16868 -0.024201

결과가 위의 결과와 일치한다는 것을 볼수있다.

비교하기 쉽게 그래프를 그려보겠다.

par(mfrow = c(1, 3))
plot(princomp(Dat[, 1:4])$score[, 1:2], main = "princomp()", xlab = "PC1", ylab = "PC2")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)
plot(cmdscale(dist(Dat[, 1:4]), k = 4)[, 1:2], main = "cmdscale()", xlab = "PC1", 
    ylab = "PC2")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)
plot(ConvD[, 1:2], main = "manual", xlab = "PC1", ylab = "PC2")
abline(h = 0, lty = 2)
abline(v = 0, lty = 2)
