기본 그래핑

R에서 그래프를 그릴때 사용되는 팩키지들은 여러가지가 있으며, 또 기본 그래프보다 더 많은 기능과 스타일을 제공한다. 하지만 기본 그래프도 많은 가지수의 그래프 종류를 제공하고있고, 또 제공되는 코딩을 잘 활용하면 고차원적인 그래프를 그리는것도 가능하다. 아래의 함수로 기본 그래프의 몇가지 샘플 데모를 볼수있다.

demo(graphics)

그래프를 그릴때 plot()이라는 함수를 사용하는데 데이터의 변수와 지정해준 x, y축들에 따라서 자동으로 그래프 스타일을 바꾸어주기도한다.

2개의 수치 변수만 있는 데이터일 경우

plot(cars)

1개의 범주형 변수와 1개의 수치변수를 지정할경우

plot(breaks ~ tension, warpbreaks)

2개이상의 변수가있는 데이터에 x, y 축을 정해주지 않았을때

plot(LifeCycleSavings)

그래핑 요소들을 지정할때

우선 아래의 기본 그래프가 그래핑 요소를 바꿀때 어떻게 변하는지 설명하겠다.

x <- c(0:100)
y <- 1/x
plot(x, y)

그래프의 점들을 다른 모양이나 스타일로 정할때

pch를 0 에서 25 사이의 번호를 줄수도있고, 원하는 심벌을 정해줄수도있다. 

plot(x, y, pch = "+")


plot(x, y, pch = 25)

cex (character expansion)를 이용하여 점의 크기를 바꿀수있다.

plot(x, y, pch = 25, cex = 2)

선을 사용할때

점선은 lty 를 사용하여 원하는 모양의 점선으로 지정한다. 여기서 1은 실선이며 2이상은 각기 다른 모양의 점선들이다.

plot(x, y, type = "l", lty = 1)


plot(x, y, type = "l", lty = 3)

lwd는 선의 굵기를 정해준다.

plot(x, y, type = "l", lty = 1, lwd = 3)

색을 바꿀때

col을 이용하여 번호를 입력하거나 색깔의 이름을 직접 적어준다.

plot(x, y, type = "p", pch = 16, cex = 1.5, col = 2)


plot(x, y, type = "l", lty = 1, lwd = 2, col = 3)


plot(x, y, type = "l", lty = 3, lwd = 1, col = "blue")

제목과 부제목을 삽입할때

기본으로 제목(main)은 위에 부제목(sub)은 맨아래에 표시된다. 

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, main = "Test", sub = "Hyperbola")

x, y 축에 이름을 삽입할때

xlab은 x축의 이름. ylab은 y축의 이름

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, xlab = "X", ylab = "1/X", 
    main = "Test", sub = "Hyperbola")

제목과 부제목에 색을 넣을때

col.main는 제목의 색깔. col.sub는 부제목의 색깔.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.main = 3, col.sub = 6, 
    xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola")

x, y축의 이름과 단위에 색을 넣을때

축의 이름은 col.lab을 축의 단위는 col.axis를 사용하여 색을 지정해준다.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola")

텍스트 크기 바꾸기

제목은 cex.main, 부제목은 cex.sub, 축의 이름은 cex.lab 그리고 축의 단위는 cex.axis를 사용하여 크기를 지정해준다.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola")

글꼴 바꾸기

제목, 부제목, 축의 이름과 축의 단위의 글꼴(font)들도 아래와 같이 바꾸어줄수 있다. 

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3)

텍스트 행 맞추기

adj를 사용하여 그래프 제목, 부제목, 축 이름과 단위의 행을 조절할수있다. 0=왼쪽으로 맞추기, 1=오른쪽으로 맞추기, 0.5=중앙. 0과 1사이의 소숫점 값으로 그 외 위치를 조절한다.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7)

축의 단위 조절

lab를 이용하여 축의 단위가 보여지는 포맷을 바꿀수있다. lab=c(x축의 단위, y축의 단위, 보여지는 단위의 간격)

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10))

축의 단위의 방향 바꾸기

las를 이용하여 축의 단위표시의 방향을 바꾸어준다. 0=축과 평행, 1=가로 방향, 2=세로 방향

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10), las = 2)

축이름과 단위의 거리조절

mgp를 사용하여 축이름과 단위가 축에서 얼마나 떨어진 더리에 표시되는지 지정해준다. mgp=c(축의 이름과 축의 거리, 축의 단위와 축의 거리, 축과 그래프 면의 거리)

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10), las = 2, mgp = c(3, 2, 1))

단위 표시점 (tick mark)의 길이

tck를 이영하여 단위 표시점의 길이를 조절한다. tck값은 그래프 면적의 분수로 삽입한다. 여기서 그리드 라인의 값은 1로 기준하며, 표시점이 그래프안으로 보여질 경우는 양수로 밖으로 보여질경우는 음수로 삽입한다. 

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10), las = 2, mgp = c(3, 2, 1), tck = 0.01)

그래프 면적 바꾸기

기본으로 x, y축이 데이터 값의 범위보다 조금 더 크게 잡아서 여백을 남기는데 xaxs와 yaxs를 사용하여 여백을 없애 버릴수도있다. r=여백허용. i=여백불허.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10), las = 2, mgp = c(3, 2, 1), tck = 0.01, xaxs = "i", yaxs = "r")

마진 설정

mai=c(아랬쪽 마진, 왼쪽마진, 윗쪽 마진, 오른쪽 마진).

mar은 mai와 같으나 마진의 측정값을 텍스트라인으로 정한다.

plot(x, y, type = "l", lty = 1, lwd = 3, col = 2, col.axis = 4, col.lab = 5, 
    col.main = 3, col.sub = 6, cex.axis = 0.6, cex.main = 1.5, cex.sub = 1, 
    cex.lab = 0.8, xlab = "X", ylab = "1/X", main = "Test", sub = "Hyperbola", 
    font = 2, font.axis = 2, font.lab = 1, font.main = 4, font.sub = 3, adj = 0.7, 
    lab = c(11, 3, 10), las = 2, mgp = c(3, 2, 1), tck = 0.01, xaxs = "i", yaxs = "r", 
    mai = c(2, 1, 3, 0))
