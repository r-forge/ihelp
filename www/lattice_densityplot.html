여기서는 고등의 그래프를 그리는 팩키지중 하나인 lattice 팩키에서 밀도를 그래프로 나타내주는 densityplot() 함수를 소개하고자 한다. 함수내에 사용되는 매개변수들은 xyplot()과 거의 동일하다. 다른점은 densityplot 그래프의 특성상 xyplot에서 사용되는 type과 같은 매개변수는 무의미하며, 유의할점은 x축이 수치형 변수이어야 한다는 점이다.

아래는 기본적인 densityplot 그래프이다.

library(lattice)

densityplot(~yield, barley)

그래픽요소를 바꾸어 주는것과 제목및 부제목을 입력하는것은 xyplot()함수와 같다. 

densityplot(~yield, barley, lty = 3, lwd = 2, col = "violet", main = "Amount of Barley Yield", 
    sub = "보리 수확양", par.settings = list(par.main.text = list(cex = 1.8, 
        col = "blue"), par.sub.text = list(cex = 1.2, col = "green"), par.xlab.text = list(cex = 0.8, 
        col = "purple"), par.ylab.text = list(cex = 0.8, col = "steelblue")))

다수의 창(pane)을 사용해서 densityplot 그리기

xyplot()과 마찬가지로 하나의 데이터소스안에서 특정 변수를 기준으로 동시에 변수값과 일치하는 다수의 창에서 다른 그래프를 출력시킬때 아래와 같이한다. xyplot과 마찬가지로 만약 수치형변수를 기준으로 창을 나눌때는 as.character()를 사용하여 범주형변수로 먼저 바꿔준다.

densityplot(~yield | variety, barley)

위의 그래프에서 창의 위치를 정리해주려면 layout 매개변수를 사용한다. 

densityplot(~yield | variety, barley, layout = c(2, 5))

그래프내에서 그룹을 정해줄때

데이터안에서 특정 변수값을 그룹으로 정한후 그래프를 출력할때는 xyplot() 처럼 아래와 같이 groups를 정해준다. 여기서 groups로 정해진 변수는 그래프내에서 자동으로 factor로 인식이 되므로 다른 변수로 점의 크기나 아니면 다른 매개변수들을 지정하였을때 이점에 유의하여야 한다. 

densityplot(~yield | variety, barley, layout = c(2, 5), groups = barley$year)

위의 그래프에서 그룹으로 정해진 변수값들을 기호일람표로 표시할때 아래와 같이한다. 여기서 size는 기호의 크기이고 cex는 설명문의 글씨크기이다, 그리고 between은 기호와 설명문 사이의 간격이다. 

densityplot(~yield | variety, barley, layout = c(2, 5), groups = barley$year, 
    auto.key = list(space = "right", cex = 0.7, size = 2, lines = TRUE, between = 0.5))

밀도를 계산할때 지정해주어야 하는것이 면적의 단위인데 이 그래프에서는 x축을 기준으로한 간격이 밀도를 측정하는 일종의 면적단위가된다. 그래프내에서 이것을 정하는 매개변수는 n으로 기본값은 50이다. 아래는 위의 그래프의 n값을 10으로 바꾸었을 경우의 예이다. 그래프의 선이 부드럽지 못하고 각이져진것을 볼수있다.

densityplot(~yield | variety, barley, layout = c(2, 5), groups = barley$year, 
    auto.key = list(space = "right", cex = 0.7, size = 2, lines = TRUE, between = 0.5), 
    n = 10)
