한개 이상의 그래프를 동시에 나열하기

한개이상의 그래프를 비교하고 싶을때 mfrow()나 mfcol()함수를 사용하여 동시에 그래프들을 같은 그래픽 디바이스안에 출력할수있다.

x <- c(0:100)
y <- 1/x
z <- x^2 + x
w <- sqrt(x)

3열 2행으로 그래프들을 나열할때 mfcol()을 사용하여 다음과 같이할수있다.

par(mfcol = c(3, 2))
plot(x, y)
plot(x, z)
plot(x, w)
plot(x, y, type = "l", lty = 1, lwd = 2, col = 2)
plot(x, z, type = "l", lty = 1, lwd = 2, col = 3)
plot(x, w, type = "l", lty = 1, lwd = 2, col = 9)

아래는 mfrow()를 사용하여 그래프들을 2열 4행을로 나열하였다.

par(mfrow = c(2, 4))
plot(x, y)
plot(x, z)
plot(x, w)
plot(x, x)
plot(x, y, type = "l", lty = 1, lwd = 2, col = 2)
plot(x, z, type = "l", lty = 1, lwd = 2, col = 3)
plot(x, w, type = "l", lty = 1, lwd = 2, col = 9)
plot(x, x, type = "l", lty = 1, lwd = 2, col = 5)

새창에 그래프 그리기

Windows를 사용하는경우 기존 출력한 그래프창을 놔두고 새창에 다른 그래프를 출력할때 x11()나 windows() 를 사용한다.

Unix에서는 x11(), OS X에서는 quartz()를 사용한다.

x11()
plot(1:10)
windows()
plot(rnorm(10))

기존 출력된 그래프에 다른 그래프 추가하기

기존 출력된 그래프위에 겹치도록 다른 그래프를 그리고 싶을때 아래와 같이한다.

x <- 1:10
par(oma = c(1, 2, 1, 3))
plot(1/x ~ x, type = "l", col = "blue", ylim = c(0, 1))
par(new = TRUE)
plot(x/10 ~ x, type = "l", col = "red", yaxt = "n", ylab = "")
axis(4, at = seq(0, 1, by = 0.2), pos = c(10.4, 0), col = "black", lty = 1)
mtext("x/10", 4, line = 3)

그래프를 파일로 출력시킬때

그래프를 그래프창으로 출력시키지않고 pdf파일로 출력시킬때 아래와같이한다. 실행후 항상 dev.off()를 사용하여 출력 디바이스를 종료시킨다.

x<-1:100
pdf("파일이름.pdf", width=10, height=8)
plot(x^2~x,type="l")
dev.off()

다른 포맷의 파일로 저장할때 아래의 옵션들이있다.

jpg는 jpeg();
png는 png();
bmp는 bmp();
tif는 tiff()

사용시 파일 이름의 extension을 사용하는 함수와 일관되게 바꿔준다.
