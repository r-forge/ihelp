초보들을 위한 RStudio안에서 knitr 팩키지와 markdown을 사용하여 html

문서를 생성하는 방법

우선 RStudio를 실행한후 knitr팩키지를 설치한다. 그리고 파일메뉴에서 새로운 문서로 R Markdown을 선택한다. 아래의 화면처럼 간단하게 마우스로 클릭만 해주면 된다.

R markdown을 선택하면 예시가 포함된 R Markdown 문서가 열린다. (여기에서 R Markdown을 줄인 말인 .Rmd가 문서의 확장자임을 기억하면 유용합니다).

새로열린 R Markdown 문서 상단에 새로운 radio button 들이 있는데 그중 MD라고 쓰여있는 버튼을 누르면 우측하단에있는 help 페이지에 R Markdown을 작성하는데 필요한 명령어와 포맷등등이 설명되어있는 페이지가 뜬다.

미리 작성되어 있는 예시와 help page를 이용하여 손쉽게 rmd 파일을 작성한후 저장한다.

그리고 위의 캡쳐된 화면에서 보이듯이 radio button들중 knit HTML이라는 버튼을 누르면 작성한 rmd 파일에 해당하는 html문서가 작성되어 미리보기가 뜨며 동시에 rmd파일과 같은 장소에 같은 이름으로 저장된다.

명령어라인을 이용하여 R을 사용하는 경우

유닉스와 같은 컴퓨팅 환경 또는 윈도우즈에서 도스 상태에서 Rmd 파일으로부터 HTML 문서를 생성하고자 하는 경우에는 아래와 같이 입력합니다.

Rscript -e "knitr:::knit2html('filename')"

명령어가 실행된 디렉토리내에 filename.md 라는 파일이 생성된 후 filename.html 문서가 생성되어 있음을 확인할 수 있습니다.
