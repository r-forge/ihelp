K-평균 군집분석과 주성분 분석의 교잡 (Hybridisation)

K-평균 군집분석을할때 가장 중요한것중 하나가 주어진 데이터안에서 군집이 얼마나 잘 형성되느냐이다. 어떤 경우 군집형성이 잘 안되어서 다른 알고리즘을 찾는 수도있고, 어떤 경우에는 데이터내에 존재하는 이상치들을 먼저 제거한후 군집을 형성시킬수도 있다.

여기서는 데이터를 잃지않고 변수들을 변형시켜서 군집형성이 더 잘되게 도와주는 방법을 소개하려한다. 그 방법으로 주성분 분석을 응용하였는데, 흔히 주성분 분석을 사용하는 이유는 차원축소를 하기 위함이다. 여기서는 단순한 차원축소를 뛰어넘어서 군집들이 더 밀집되게 하고 이상치는 더 선명하게 떨어지게하는 수단으로 주성분 추출중에 변수를 정규화 (normalise)시키는 방법을 사용한다.

여기서 설명하는 방법은 아래의 논문에서 제시된 방법을 응용하였음을 먼저 밝힌다.

A hybridized K-means clustering approach for high dimensional dataset; Rajashree Dash, Debahuti Mishra, Amiya Kumar Rath, Milu Acharya; International Journal of Engineering, Science and Technology Vol. 2, No. 2, 2010, pp. 59-66; http://ijest-ng.com/ijest-ng-vol2-no2-pp59-66.pdf

우선 이해를 쉽게하기위해 아이리스 데이터를 사용해 이 데이터 점들의 분포도를 주성분 분석 적용 전과후를 비교하겠다. 여기서는 일반적인 주성분 분석으로 변수들을 정규화시키지 않았다.

Sys.setlocale("LC_CTYPE", "Korean_Korea")

## [1] "Korean_Korea.949"


Dat <- iris[, 1:4]

CTR <- aggregate(. ~ Species, iris, mean)
km2 <- kmeans(Dat, centers = CTR[, 2:5], iter.max = 100)

PC <- princomp(Dat)

PC2 <- as.data.frame(PC$score[, 1:2])
PC2$Species <- iris$Species
CTR_PC2 <- aggregate(. ~ Species, PC2, mean)
kmPC <- kmeans(PC2[, 1:2], centers = CTR_PC2[, 2:3], iter.max = 100)

여기서 주성분 분석으로 데이터를 변형한후 K-평균 군집분석을 할경우 아래의 그래프에서 보이듯 군집의 형성이 더 명확해지는것을 관찰할수있다. 맨 오른쪽 그래프가 주성분 분석을 이용하여 생성된 군집들인데 2차원 그래프임에도 겹치는곳이 거의 없이 명확하게 군집들이 나누어진것을 볼수있다. 이것은 제1, 제2 주성분축들이 아이리스 데이터의 4개의 변수들을 잘 반영해주고 있다는 뜻이다.

정규화 과정 추가

아래는 주성분 추출시 변수들을 정규화시키는 과정을 추가한것이다.

보통 여기서 몇개의 주성분축을 사용할지 정할때, 상응하는 eigen value가 평균값보다 높은 주성분 축들을 사용하게 한다. 아래의 예에서는 2개로 미리 정해주었는데, 그 이유는 아이리스 데이터안에 변수가 4개 밖에없는 관계로 주성분 축이 2개 미만으로 선택될수있기 때문이다. 

V <- apply(Dat, 2, var)
Input <- Dat[, which(V > 0)]

Mean <- apply(Input, 2, mean)
Sdev <- apply(Input, 2, sd)

Adj.Data <- t(apply(Input, 1, function(x) (x - Mean)))
Norm.Data <- t(apply(Adj.Data, 1, function(x) (x/Sdev)))

Cov <- var(Norm.Data)

Eig.Vec <- eigen(Cov)
Featured <- Eig.Vec[[2]][, 1:2]
temp <- t(Featured) %*% t(Norm.Data)
test1 <- t(temp)

DatN <- cbind(as.data.frame(test1), Species = iris$Species)
CTR_PCN <- aggregate(. ~ Species, DatN, mean)
kmPCN <- kmeans(test1, centers = CTR_PCN[, 2:3], iter.max = 100)

아래는 데이터의 원형과 주성분 값을 이용한 군집 분석, 그리고 정규화된 변수들을 이용해서 주성분을 추출한후 군집 분석한 결과들을 비교하기위하여 데이터를 준비하는 과정이다.

colnames(Dat) <- paste("Col", c(1:4), sep = "")
Dat$Species <- iris$Species
Dat$Cluster <- km2$cluster
Dat$Dist <- apply(Dat, 1, function(x) sqrt(sum((as.numeric(x[1:4]) - km2$center[as.numeric(x[6]), 
    ])^2)))
Dat$Ind <- "Orig"

colnames(PC2)[1:2] <- paste("Col", c(1:2), sep = "")
PC2$Cluster <- kmPC$cluster
PC2$Dist <- apply(PC2, 1, function(x) sqrt(sum((as.numeric(x[1:2]) - kmPC$center[as.numeric(x[4]), 
    ])^2)))
PC2$Col3 <- NA
PC2$Col4 <- NA
PC2$Ind <- "PCA-2D"

colnames(DatN)[1:2] <- paste("Col", c(1:2), sep = "")
DatN$Cluster <- kmPCN$cluster
DatN$Dist <- apply(DatN, 1, function(x) sqrt(sum((as.numeric(x[1:2]) - kmPCN$center[as.numeric(x[4]), 
    ])^2)))
DatN$Col3 <- NA
DatN$Col4 <- NA
DatN$Ind <- "PCA-Norm"

DatX <- rbind(Dat, PC2, DatN)
DatX1 <- split(DatX, DatX$Ind)
DatX1 <- lapply(DatX1, function(x) {
    x$Dist <- x$Dist/max(x$Dist)
    x
})
DatX1 <- ldply(DatX1, data.frame)

아래의 그래프에서 보이듯 정규화 변수를 이용한 k-평균 군집분석과 주성분 분석의 교잡이 가장 밀도가 높고 명확한 군집들을 형성시키는것을 관찰할수있다. 그리고 이상치들이 더욱 뚜렸해지는 것도 볼수있다.
